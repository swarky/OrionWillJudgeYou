# What is Orion Will Judge You

OWJY, for short, is a Chrome web browser extension that will display a picture of my cat whnever youy spend too much time on certain websites, such as Twitter, Reddit and such.

Once Orion appears, the only way to gently ask him to leave is to close the tab from the aformentionned website. Only Then Orion will go away.


## Features
- Stop brainrotting as orion will reminds you that endless scrolling is bad for you.
- You can customize the websites in the extension settings, add and remove them.
- You can customize the timer
